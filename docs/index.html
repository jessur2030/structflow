StructFlow

A local-first JSON viewer, code editor, and snippet library that lives in your browser's side panel.

Edit on one syntax-highlighted surface with auto-detect on paste, beautify 8 languages and highlight 20+ more, explore JSON as a tree, snapshot code to a PNG, and keep a searchable library of notes and snippets — all offline. Nothing you write leaves your device.
