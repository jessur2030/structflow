Support

Need help with StructFlow, or found a bug? The fastest way to reach us is GitHub issues.

Open an issue on GitHub →

StructFlow runs entirely in your browser and your snippets never leave your device, so please don't paste private data when asking for help.

What to include

  • Browser and version.
  • Extension version (see Settings → About).
  • What you were trying to format or view.
  • The smallest non-sensitive sample that reproduces the issue.
  • A screenshot if the issue is visual.

Please don't include private API keys, tokens, customer data, or production payloads.
