# Privacy Policy

Effective date: 2026-06-27

StructFlow is a local-first Chrome extension for formatting code, viewing JSON, taking notes, and saving snippets.

## Data Collection

StructFlow does not collect, transmit, sell, or share personal data.

The extension does not use analytics, tracking pixels, remote logging, advertising identifiers, or AI services.

## Local Storage

StructFlow stores app data locally in your browser:

- Saved library entries, including raw input, formatted output, titles, tags, project assignments, pinned state, and timestamps.
- Projects/folders and their display colors.
- Formatter draft content.
- Preferences: app theme, syntax theme, default language, recently used languages, and the in-page JSON viewer on/off setting.
- Temporary selected text sent from the context menu into the side panel.

This data stays on your device unless you manually export it.

## Permissions

StructFlow requests these Chrome extension permissions:

- `sidePanel`: shows the app in Chrome's side panel.
- `storage`: stores extension preferences and temporary context-menu input.
- `contextMenus`: adds a right-click action for formatting selected text.
- Content script access on pages: detects raw JSON pages and renders the in-page JSON viewer.

## Network Use

StructFlow does not make network requests from the extension for formatting, storage, analytics, or licensing.

## Exported Files

When you export data, StructFlow creates files locally in your browser and downloads them to your device. Exported ZIP files may contain saved snippets, notes, titles, tags, project names, and metadata.
